Chat Embedding Demo

Embed Configuration

Configure the embed URL below and click Apply to update the chat iframe source.

Usage Examples

Minimal

<iframe
  src="http://localhost:5173/embed/chat/abc-123?tenantId=tenant-456"
  style="width:400px; height:600px; border:none; border-radius:12px;"
  allow="clipboard-write"
></iframe>

With theme & context data

<iframe
  src="http://localhost:5173/embed/chat/abc-123?tenantId=tenant-456&theme=dark&ctx_userId=user-789"
  style="width:100%; height:600px; border:none;"
  allow="clipboard-write"
></iframe>

URL parameters

ParamRequiredDescription
tenantIdYesTenant the chat agent belongs to
themeNolight or dark (default: auto)
langNoUI language, e.g. en-US
ctx_*NoContext data forwarded to the agent
